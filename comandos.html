Comandos básicos do Ubuntu

Aprender os comandos básicos do Ubuntu é essencial para gerenciar servidores e sistemas Linux de forma eficiente. Alguns comandos úteis incluem:

  • ls - Lista os arquivos e diretórios no diretório atual.
  • cd - Navega entre diretórios.
  • mkdir - Cria um novo diretório.
  • rm - Remove arquivos ou diretórios.
  • apt-get - Gerencia pacotes (instala, remove ou atualiza).
  • sudo - Executa comandos com privilégios de administrador.
  • pwd - Mostra o diretório atual.
